Objetivo

El objetivo de esta página es explicar de forma clara qué son las herramientas de diseño y para qué sirven. Este contenido fue preparado para la materia y para mostrar un ejemplo práctico.

Herramientas populares

  • Canva: Plataforma fácil para crear posters, presentaciones y diseños rápidos. Tiene plantillas listas y muchas imágenes. Canva es gratis, ¡aproveche profesor Roque!
  • Figma: Usada para diseñar interfaces y prototipos. Es colaborativa —varias personas pueden editar al mismo tiempo.
  • Adobe XD: Similar a Figma, buena para prototipado si ya usas productos Adobe.
  • Webflow: Permite diseñar de forma visual y convertir el diseño en una página web sin escribir código avanzado.
  • Google Slides / PowerPoint: No son solo para presentaciones; sirven para hacer diseños sencillos y rápidos.

Consejos prácticos

Al trabajar con estas herramientas, recuerda:

  • Mantén pocos colores (2 o 3) para que el diseño se vea limpio.
  • Usa tipografías legibles; una para títulos y otra para texto si quieres.
  • Prueba siempre cómo se ve en celular. Muchos trabajos se ven ahí primero.
  • Guarda versiones del archivo mientras trabajas para no perder cambios.
